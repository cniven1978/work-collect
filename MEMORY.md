## 微信公众号抓取方案

> 2026-06-16 更新：从 pending 状态升级为 active，引入 RSSHub 中间层

### 方案概述

微信公众号是封闭生态，无法像普通网站直接爬取。解决方案：

**自建 RSSHub (Docker) → 搜狗微信搜索/wechat2rss → 标准 RSS → 每日简报读取**

### 关键配置

| 配置项 | 值 |
|-------|-----|
| RSSHub 基地址 | http://localhost:1200 |
| 默认抓取方式 | sogou（搜狗微信搜索） |
| 降级顺序 | sogou → wechat2rss → ershcimi |
| 请求间隔 | ≥ 30秒 |
| 缓存时间 | 2小时 |

### 获取公众号微信号和 __biz ID 的方法

1. 打开 https://weixin.sogou.com 搜索公众号名称
2. 点击搜索结果中的公众号名，进入公众号主页
3. URL 中 `__biz` 参数就是公众号 ID
4. 公众号微信号在公众号主页可见

### 已知问题

- 搜狗反爬会触发验证码，需定期更新 Cookie
- RSSHub 需要保持 Docker 容器运行
- feeddd.org 已下线，不可用
- wechat2rss.xlab.app 作为备用源可用
